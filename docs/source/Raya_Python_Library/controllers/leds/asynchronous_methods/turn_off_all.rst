=================
turn_off_all()
=================

This function turns off all of the groups.

Reference
===========

Arguments
-----------

``None``

Return
---------

``None``

Exceptions
------------

See the :ref:`complete led's exceptions <leds_exceptions>`.

Usage example
================

Turn off all of the leds.

Code:

.. code:: python

   self.leds.turn_off_all()
