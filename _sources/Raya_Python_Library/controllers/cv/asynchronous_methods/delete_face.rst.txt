=======================
delete_face() 
=======================

.. raw:: html

    <hr/>

Deletes faces from a facial recognition system.

Reference
===========

Arguments
-----------

+-----------------------+-----------------------+-----------------------+
| Arguments             | Type                  | Description           |
+=======================+=======================+=======================+
| faces_to_delete       | ``list``              | List of faces to      |
|                       |                       | delete.               |
+-----------------------+-----------------------+-----------------------+
| model_name            | ``str``               | Model name.           |
+-----------------------+-----------------------+-----------------------+

Return
--------

+-----------------------------------+-----------------------------------+
| Type                              | Description                       |
+===================================+===================================+
| ``list``                          | List with face deleted.           |
+-----------------------------------+-----------------------------------+
| ``list``                          | List with face not deleted.       |
+-----------------------------------+-----------------------------------+

Exceptions
------------

``None``

See the :ref:`complete list of cv exceptions <cv_exceptions>`.
