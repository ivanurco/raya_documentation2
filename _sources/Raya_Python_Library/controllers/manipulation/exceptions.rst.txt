====================
Exceptions
====================

    <hr/>

========================================= =======================================
  Exception                                Condition

``RayaManipulationException``              Exception from manipulation method.
``RayaManipulationNotManipulation``        Not manipulation available.
``RayaManipulationAlreadyEnabled``         Manipulation is already running.
``RayaManipulationNotArmName``             Arm name is not valid.
``RayaManipulationObjNotFound``            Object to manipulate not found.
``RayaManipulationNotDetections``          Detections are not arriving.
``RayaManipulationProcessFail``            Process error.
``RayaManipulationTopicNotPublishing``     Topic is not publishing.
``RayaManipulationSrvNotAvailable``        Service is not available.
``RayaManipulationPickSolutionNotFound``   Pick solution not found.
``RayaManipulationNotReference``           Not reference.
``RayaManipulationNotArm``                 Arm not available.
``RayaManipulationNotHeight``              Not height setted.
``RayaManipulationNotTag``                 Not tag setted.
``RayaManipulationPlaceSolution``          NotFound Place solution not found.
``RayaManipulationInvalidPoint``           Point invalid.
``RayaManipulationNotPlaceObject``         Object to place not found.
