============================
remove_constraints()
============================

.. raw:: html

    <hr/>

Remove the constraint to the arm.

Reference
===========

Arguments
-----------

========= ======= ===================================
Arguments Type    Description
========= ======= ===================================
arm       ``str`` The arm to remove constraints from.
========= ======= ===================================

Return
--------

``None``

See the :ref:`complete list of arms exceptions <arms_exceptions>`
